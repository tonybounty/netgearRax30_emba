			with their associated and monitored Clients
	slavelist	Displays all Blanket Slaves

	monitorlist	Lists all monitored STAs by this Slave
	monitoradd	Adds a STA to be monitored by this Slave
			Ex : wb_cli -s monitoradd -a mac -b bssid
	monitordel	Deletes a STA from monitored if MAC is provided,
			else deletes all STAs from monitor
	weakclient	Makes a client weak
	weakcancel	Makes a weak client normal
	msglevel	Set Message Level
