       - chip_ser_num       :Specify a unique chip serial number. This number is only readable via OTP
       - jtag_pwd           :Specify a JTAG unlock password. To be used when JTAG interface is locked via OTP
       - jtag_pwd_rdlock    :readlocks the JTAG password, JTAG password can then not be read via OTP
       - jtag_lock          :password locks the JTAG interface.
                             JTAG can then only be unlocked via debugger by using the configured JTAG password
       - jtag_perma_lock    :permanently locks the JTAG interface. JTAG can then never be unlocked 
       - otp_cust_lock      :permanently read/write locks the JTAG config OTP region
